JBC

This Java Boot Camp (JBC) is a web development boot camp sponsored by the Department of Labor and implemented by Montgomery College. It's an intensive eight-week course ranging in topics from core Java to HTML and CSS. Once a week, we complete an individual coding challenge from start to finish in seven hours. In our final week we will complete a group challenge in which we will design a fully dynamic web site, complete with our own back-end code and our own front-end wizardry.
