Java Basics

Primitive Data Types

These are data types that are built in to the Java system. For more on primitive data types see the official tutorial.

int and long

The int data type allows values between -2,147,483,648 (negative 2 to the 31st power) and 2,147,483,647 (positive 2 to the 31st power, minus 1) inclusive. In other words, I believe this means it is using 32-bit precision but reserving one bit for the sign. long is the 64-bit version of int.

float and double

float is 32-bit while double is 64-bit. Typeically, double is the one you would use by default, but use float if you need to save memory in large arrays of floating point numbers. While these data types do have a limit on their range of values, it's not something we would typically encounter in daily usage! However, these should never be used for exact Decimal calculations, such as for currency or basic math! For this purpose, you would need to use the java.math.BigDecimal class.

boolean

The boolean data type can be true or false. Note that the true and false are not capitalized in Java!

char
